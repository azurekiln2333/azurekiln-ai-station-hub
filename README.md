# AzureKiln AI Hub

AI 中转站汇总站，基于提供的 NexusAI UI demo 重构为可运行的 React/Vite 单页应用。

## 功能

- AI 中转站目录、搜索、分类筛选和多标签筛选
- 中转站卡片收藏、详情页、性能概览、模型兼容、安全能力和价格信息
- 点击直达外部中转站或文档
- Demo 登录、本地收藏列表、最近访问记录
- 响应式桌面和移动端导航

## 运行

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```
